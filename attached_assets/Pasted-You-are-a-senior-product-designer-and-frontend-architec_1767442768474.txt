You are a senior product designer and frontend architect.

Build a COMPLETE, PROFESSIONAL, SaaS-STYLE FRONTEND for a
MARKETING AGENCY STRATEGY & CAMPAIGN AUTOMATION PLATFORM.

This is NOT a simple marketing website.
It is a multi-page, multi-workflow web application.

The frontend must prioritize:
• Clarity
• Speed
• Workflow-based navigation
• Enterprise-grade polish

====================================================
GLOBAL DESIGN SYSTEM
====================================================
• Clean modern SaaS aesthetic
• White / light background with subtle neutral grays
• One accent color (blue / violet / teal)
• Minimal gradients (only for highlights)
• No heavy illustrations
• Use cards, tables, and step-based layouts
• Smooth but subtle micro-animations
• Typography: strong hierarchy, readable body text
• Desktop-first (responsive later)

====================================================
LAYOUT & NAVIGATION
====================================================
• Public website + Auth + App
• Sidebar navigation inside the app
• Top bar for account actions
• Clear breadcrumbs inside client workspace
